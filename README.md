# IA na Sala de Aula

Landing page desenvolvida para uma oficina destinada a professores sobre o uso do Gemini e NotebookLM na educação.

## Tecnologias
- HTML5
- CSS3
- Design Responsivo

## Objetivo
Apresentar, de forma prática e visual, como ferramentas de Inteligência Artificial podem auxiliar professores no planejamento de aulas, produção de materiais, avaliações e organização do ensino.
